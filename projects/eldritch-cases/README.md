# Eldritch Cases

A roguelike deckbuilder combining XCOM's strategic roster management with Slay the Spire's tactical card combat, set in 1920s Lovecraftian horror.

## Concept

You run a paranormal investigation agency. Select cases from an escalating supernatural crisis. Deploy teams of 3 investigators whose decks merge into a single draw pile for Slay the Spire style combat. Investigators die permanently. The doom clock ticks. Save the world before time runs out.

**Target run length:** 2 hours
